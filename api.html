  • API

Что-бы запустить наше api без настроек вам нужно перейти по алресу: passwdgen.ru/get_passwd.php. Если вам нужно опредиленное количество символов то перейдите на этот адрес: passwdgen.ru/get_passwd.php?symbols=10. После symbols= надо поставить число длину пароля которую вам нужно.
